# Warehouse-Management-Database
This project records everything about the database design.

The whole database design is in the Warehouse Management Database folder.

Please download it and run the interface.py.

Please have your MySQL server activated.

Please open the Interface.py and modify the password in the beginning of the interface to connect to the server mentioned above.

Please run the program and put in the labels (e.g. 1-1) shown on console log to execute operations.

Main includes the record of our process.

# Python Library Requirement

pandas 2.3.3
mysql-connector-python 9.5.0
